# DownloadSchedulerApp
Demo apllication that fetches the data from API using retrofit and displays the data in recycler view with a pie chart on the top.

#### Concepts Used
1. MVVM
2. Retrofit
3. Room Db
4. Pie Chart
5. Singleton Class
